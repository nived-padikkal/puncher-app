# 🟢 Simple UI First React App (with Expo)

A **React Native app** built using **Expo**, designed with a simple UI. It includes basic **login form**, partial **location access**, and **local storage** features. This app serves as a prototype and is currently under development. Backend integration and full functionality will be added soon.
